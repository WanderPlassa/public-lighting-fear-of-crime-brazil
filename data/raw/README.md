# Raw Data

This repository does not redistribute the original data used in the analyses.

## Dataset

The analyses are based on the **2012 Brazilian National Victimization Survey (PNV 2012)**.

Information about the survey is available from the Brazilian Ministry of Justice:

https://www.gov.br/mj/pt-br/assuntos/sua-seguranca/seguranca-publica/analise-e-pesquisa/download/pesquisa-vitimizacao

## Data access

Researchers are responsible for obtaining access to the original dataset.

The replication scripts expect the dataset to be available in this folder with the filename:

pm3549_tipo1_v1.rds

The original data are **not included** in this repository.
