# Raw Data

This repository does not redistribute the original survey data used in the analyses.

## Dataset

The study is based on the **2012 Brazilian National Victimization Survey (PNV 2012)**.

## Data access

Information about the survey is available from the Brazilian Ministry of Justice:

https://www.gov.br/mj/pt-br/assuntos/sua-seguranca/seguranca-publica/analise-e-pesquisa/download/pesquisa-vitimizacao

Researchers should obtain the original dataset from the Ministry of Justice (or the institution responsible for providing access to the PNV microdata).

After obtaining the data, place the file

pm3549_tipo1_v1.rds

inside this folder:

data/raw/

The replication scripts assume this filename and directory structure.
