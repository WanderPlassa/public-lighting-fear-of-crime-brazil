# Raw Data

This repository does not redistribute the original data used in the analyses.

## Dataset

The analyses are based on the **2012 Brazilian National Victimization Survey (PNV 2012)**.

The dataset was originally obtained from the **CRISP (Center for the Study of Crime and Public Security, Federal University of Minas Gerais)** data repository.

Repository:

https://www.crisp.ufmg.br/repositorio-de-dados/

## Data availability

At the time this replication package was prepared, the PNV 2012 microdata were no longer available for download from the CRISP repository.

Therefore, the original dataset is **not included** in this repository.

Researchers interested in replicating the analyses should contact the CRISP/UFMG or the institution responsible for the survey to request access to the original microdata.

The replication scripts assume that the dataset is stored as:

data/raw/pm3549_tipo1_v1.rds
